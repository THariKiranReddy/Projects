# FurniStore - Modern Furniture E-Commerce
## Introduction
FurniStore is a feature-rich React application designed to provide a seamless furniture shopping experience. It features a robust product discovery system, allowing users to filter items by brand and price. ## Key Features
### Authentication: Secure user login to access personalized shopping features.

### Advanced Filtering: * Search bar for real-time product discovery.

### Filter by Company Name.

### Sort by Price (Low to High / High to Low).

### Dynamic Shopping Cart: Add, remove, and update furniture quantities with real-time price calculations.

### Checkout Workflow: Integrated form for shipping details (Name & Address) that converts cart items into a permanent Orders List.

### Toast Notifications: Real-time feedback using react-toastify for actions like "Added to Cart" or "Order Placed."
